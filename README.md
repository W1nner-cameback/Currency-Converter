## Описание программы
*"Currency-Converter - это GUI приложение для конвертирования валют. Позволяет
конвертировать разные валюты и сохраняет историю."*

## Требования для запуска
Что нужно установить на компьютер:
- Python 3.4 или выше

## Как запустить
* git clone https://github.com/W1nner-cameback/Currency-Converter.git
* cd Currency-Converter
# Как пользоваться
* при запуске приложения необходимо ввести сумму денег, которую вы хотите конвертировать
* выбрать из какой валюты в какую
* нажать кнопку "Конвертировать", после чего вы увидете результат, т это сохранится в истории
